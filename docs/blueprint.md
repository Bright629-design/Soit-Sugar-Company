# **App Name**: Soit Sugar Company

## Core Features:

- Product Showcase: Display various types of sugar and packaging sizes with high-quality images and detailed specifications.
- Quote Request System: Enable customers to request quotes for specific products and quantities.
- Distributor Support: Provide a dedicated section for distributors with resources and support information.
- Sustainability Information: Share details about eco-friendly practices, farmer partnerships, and community impact initiatives.
- Contact Options: Offer multiple contact methods including phone, WhatsApp, email, and an inquiry form with an embedded Google map.
- AI Chatbot: Implement an AI chatbot tool using the Gemini API and RAG system to answer customer support and product-related questions.
- Admin Panel: Provide an admin login for managing products, viewing customer inquiries, and updating content.

## Style Guidelines:

- Primary color: Fresh green (#7CB342), reflecting the sugarcane fields and the company's eco-friendly focus.
- Background color: Light, desaturated green (#E8F5E9), creating a clean and modern backdrop.
- Accent color: Earthy brown (#A1887F), providing a trustworthy, industrial feel analogous to the green and suggesting the soil.
- Body and headline font: 'PT Sans', a humanist sans-serif font to balance modernity with a touch of warmth, for both headlines and body text.
- Use clean, line-style icons to represent product categories, sustainability efforts, and contact options.
- Employ a clean and modern layout with clear sections and responsive design for optimal viewing on all devices.
- Incorporate smooth animations and scrolling effects to enhance user experience without being distracting.
